# poscode-malaysia
Postcode available for Malaysia. A compact version from https://github.com/atqnp/postcode-malaysia with availability for the following column
- Poscode
- City
- Country

## Available State
- Perlis
- Kedah
- Pulau Pinang
- Perak
- Selangor
- WP Putrajaya
- WP Kuala Lumpur
- Negeri Sembilan
- Melaka
- Johor
- Pahang
- Terengganu
- Kelantan
- Sabah
- WP Labuan
- Sarawak
